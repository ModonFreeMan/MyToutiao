# 清晰度切换公开视频源记录

记录用于验证清晰度切换、多码率播放、预加载和播放恢复的公开视频源。

验证日期：2026-06-07

## 选源原则

- 优先选择 HLS master playlist，要求同一视频内容下有多个 variant。
- 优先选择普通 VOD，避免 DRM、鉴权、特殊低延迟协议，降低本地调试成本。
- 用于 mock 数据时，应尽量让每个视频的不同清晰度指向同一内容的不同 variant，而不是混用不同视频文件。
- 公开视频源可能受 CDN、跨域、证书、区域网络影响，接入前需要重新验证可访问性。

## 后续验证命令

验证 master playlist 是否可访问，并统计 variant：

```powershell
$urls = @(
  "https://d2zihajmogu5jn.cloudfront.net/sintel/master.m3u8",
  "https://d2zihajmogu5jn.cloudfront.net/big-buck-bunny/master.m3u8",
  "https://test-streams.mux.dev/x36xhzz/x36xhzz.m3u8",
  "https://test-streams.mux.dev/test_001/stream.m3u8",
  "https://devstreaming-cdn.apple.com/videos/streaming/examples/bipbop_4x3/bipbop_4x3_variant.m3u8",
  "https://demo.unified-streaming.com/k8s/features/stable/video/tears-of-steel/tears-of-steel.ism/.m3u8",
  "https://maitv-vod.lab.eyevinn.technology/VINN.mp4/master.m3u8",
  "https://maitv-vod.lab.eyevinn.technology/tearsofsteel_4k.mov/master.m3u8"
)

foreach ($url in $urls) {
  $response = Invoke-WebRequest -Uri $url -UseBasicParsing -TimeoutSec 12
  $content = if ($response.Content -is [byte[]]) {
    [System.Text.Encoding]::UTF8.GetString($response.Content)
  } else {
    [string]$response.Content
  }
  $variants = ([regex]::Matches($content, '#EXT-X-STREAM-INF')).Count
  $resolutions = ([regex]::Matches($content, 'RESOLUTION=\d+x\d+') |
    ForEach-Object { $_.Value }) -join ', '
  "$url => $($response.StatusCode), variants=$variants, $resolutions"
}
```
